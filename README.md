# ProyectoSIO
Sistema Interno Operativo (SIO)

## Descripción
Este proyecto contiene el frontend en Angular para el Sistema Interno Operativo.

## Estructura del Proyecto
- `/frontend` - Aplicación Angular

## Requisitos
- Node.js (v24.x o superior)
- npm (v11.x o superior)
- Angular CLI (v21.x)

## Instalación

### Instalar Angular CLI globalmente
```bash
npm install -g @angular/cli
```

### Instalar dependencias del frontend
```bash
cd frontend
npm install
```

## Desarrollo

### Ejecutar servidor de desarrollo
```bash
cd frontend
npm start
```

La aplicación estará disponible en `http://localhost:4200/`

### Construir para producción
```bash
cd frontend
npm run build
```

Los archivos de producción se generarán en el directorio `frontend/dist/`

### Ejecutar tests
```bash
cd frontend
npm test
```

## Tecnologías
- Angular 21.1
- TypeScript 5.9
- RxJS 7.8
- Angular Router
