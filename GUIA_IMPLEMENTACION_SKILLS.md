# Guia de Implementacion de Skills (Diseno + Programacion Limpia)

Este repo ahora incluye una skill personalizada para elevar diseno frontend y calidad de codigo fullstack.

## Archivo creado
- .github/skills/diseno-limpio-fullstack/SKILL.md

## Como usarla en Copilot Chat
Usa prompts que incluyan frases gatillo del description, por ejemplo:

- "Aplica diseno-limpio-fullstack para mejorar esta vista Angular"
- "Refactor clean code y manejo de errores con diseno-limpio-fullstack"
- "Disena este modulo con enfoque pixel perfect y clean architecture"

## Flujo recomendado por tarea
1. Pedir mini design system para la feature.
2. Pedir estructura de componentes y estados UI.
3. Implementar frontend por bloques pequenos.
4. Implementar backend con contratos API consistentes.
5. Cerrar con checklist de accesibilidad, errores y mantenibilidad.

## Prompt maestro reutilizable
"Aplica la skill diseno-limpio-fullstack en esta tarea. Entrega primero: (1) mini design system, (2) estructura de componentes/servicios, (3) implementacion por fases, (4) checklist final de UX, a11y, clean code y error handling."

## Mapeo de tus fuentes a esta implementacion
- UI/UX Pro Max: enfoque de design system + anti-patrones + checklist pre-entrega.
- Pixel Perfect: precision visual, detalle en espaciado, jerarquia y componentes.
- awesome-design-md: uso de DESIGN.md como contrato de look and feel.
- Superpowers: flujo estructurado (plan -> implementacion -> verificacion).
- wshobson agents: patrones robustos de errores, API y modelado de datos.

## Siguiente mejora opcional
Si quieres, en un siguiente paso te creo un DESIGN.md del proyecto (marca, paleta, tipografia, componentes, layout, do/dont) para que el agente mantenga coherencia visual en todas las pantallas.
